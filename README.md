# autonomous-indoor-drone

This project documents the development of an autonomous quadrotor drone capable of navigating indoors where GPS signals are unavailable.
The system combines a flight controller running ArduPilot with a Raspberry Pi companion computer that performs sensing and positioning tasks.

Project goals
*Acheive Stable flight indoors without GPS
*Use visual markers to estimate drone position
*Intergrate Companion Computer proccessing with Ardupilot navigation
